    makeView
  




  

  
Return a view-maker from view and controller functions.

  

  

    
Params

    

      view
      Function
      function to create view
    

    

      controller
      Function
      function to create controller
    

    
Returns

    

      
      Function
      function to create view against model
    

  



        		
        

function makeView(view, controller) {
  return (model, parent) => view(model, controller(model, parent));
}




        		
      

      

        

  



  

    ¶
    makeElementViews
  




  

  
Return an array of views for elements of an array-valued model.

  

  

    
Params

    

      model
      Object
      model to create element views for
    

    

      view
      Function
      view to use for elements
    

    

      controller
      Function
      function to create item controller
    

    

      parent
      Controller
      parent controller to use for item controllers
    

  



        		
        

function makeElementViews(model, view, controller, parent) {
  return model . as(model => makeView(model, view, controller, parent));
}
